## ap-csa-project
Simple Java JDBC project
* Environment: This was tested on a Mac

### Dependencies and pre-requisites
* Java11 JDK
* Postgres10
* JDBC driver
### How to compile?
````bash
javac com/myrhs/Student.java
javac com/myrhs/SchoolMain.java
````
### How to run?
* `java com.myrhs.SchoolMain`
